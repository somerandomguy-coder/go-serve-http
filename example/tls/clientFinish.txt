FIELD NAME                      SIZE (BYTES)    TYPICAL VALUE / DESCRIPTION
----------------------------------------------------------------------------------
[ TLS RECORD HEADER (ON THE WIRE) ]
Record Type                     1 byte          0x17 (Application Data - hides handshake type)
Legacy Record Version           2 bytes         0x03 0x03 (TLS 1.2 compatibility)
Record Length                   2 bytes         0x00 0x35 (Length of encrypted payload)

============================= ENCRYPTED BOUNDARY =============================
[ DECRYPTED HANDSHAKE HEADER ]
Handshake Type                  1 byte          0x14 (Finished)
Handshake Length                3 bytes         0x00 0x00 0x20 (32 bytes for SHA-256)

[ DECRYPTED FINISHED PAYLOAD ]
Verify Data                     Variable        [32 or 48 bytes of HMAC-SHA data]
                                                (Hash verification of all prior handshake bytes)

[ TLS 1.3 TRAILER ]
Inner Content Type              1 byte          0x16 (Appended inside encryption to tell the 
                                                      parser this was actually a Handshake record)
